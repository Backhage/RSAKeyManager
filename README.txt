RSAKeyManager 
-------------

The RSAKeyManager is a Windows Command Line tool that can be used to handle
RSA key pairs in Windows Key Containers.
It makes it easy to:
- Create new RSA key pairs that are automatically added to the Windows Key Storage.
- Remove any RSA key pairs from the Key Storage.
- Export RSA key pairs to XML.
- Import RSA key pairs from XML to the Key Storage.
